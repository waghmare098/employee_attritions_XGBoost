I build project to predict whether an employee is likely to leave the organization using Machine Learning techniques.
Employee attrition leads to increased recruitment costs, productivity loss, and operational instability. 
The objective of this project is to help HR departments proactively identify employees at risk of leaving and take preventive measures. 
The dataset contains employee-related information such as age, job role, salary, work-life balance, overtime status, job satisfaction 
After performing data preprocessing, feature engineering, and data analysis, an XGBoost classification model was built to predict employee attrition. 
The model was evaluated using Accuracy, Precision, Recall, F1-score, Confusion Matrix, and ROC-AUC score to ensure strong performance and accuracy .
XGBoost was selected because it provides high predictive accuracy .
Handles structured data efficiently, and offers feature importance insights
